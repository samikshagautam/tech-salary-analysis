# Tech & Data Science Salary Analysis

Exploratory data analysis of 150K+ data science job salaries using Python, Pandas, and Seaborn.

## Key Questions
- What does the overall salary distribution look like?
- How does experience level affect pay?
- Which job titles pay the most?
- Do remote roles pay differently than on-site?
- Are salaries trending up over time?

## Tools Used
- Python
- Pandas
- Matplotlib & Seaborn
- Jupyter Notebook

## Dataset
[Kaggle - Salaries for Data Science Jobs](https://www.kaggle.com/datasets/adilshamim8/salaries-for-data-science-jobs)
